## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Clone the repository or navigate to the project directory
2. Install dependencies:
```bash
npm install
```

### Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

### Building for Production

Build the optimized production bundle:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

### Deployment

#### Deploy to Vercel

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
vercel
```

#### Deploy to Netlify

1. Install Netlify CLI:
```bash
npm i -g netlify-cli
```

2. Deploy:
```bash
netlify deploy --prod
```

#### Deploy to GitHub Pages

All set up! Just run:
```bash
npm run deploy
```

Or see `DEPLOY_TO_GITHUB.md` for detailed instructions.

Quick steps:
1. Create a GitHub repository (make it Public)
2. Update remote URL: `git remote set-url origin https://github.com/YOUR_USERNAME/repo-name.git`
3. Push: `git push -u origin main`
4. Deploy: `npm run deploy`
5. Enable Pages in repository Settings

Your site will be live at `https://YOUR_USERNAME.github.io/repo-name/`

#### GitHub Pages Alternative (Manual)
1. Build the project:
```bash
npm run build
```

2. Install gh-pages (already installed):
```bash
npm install --save-dev gh-pages
```

3. Add to package.json scripts:
```json
"scripts": {
  "deploy": "gh-pages -d dist"
}
```

4. Deploy:
```bash
npm run deploy
```

## Configuration

### Tailwind CSS

The project uses Tailwind CSS v3 with custom configuration for shadcn/ui components. CSS variables are defined in `src/index.css` for theming support.

### Path Aliases

The project uses `@/` as an alias for `src/`, configured in both `vite.config.ts` and `tsconfig.app.json`.

## Mock Data

The current implementation uses mock data generated by the `useFakeStream` hook. This simulates real-time updates every 2 seconds. In production, this should be replaced with actual data streams from:

- PLC systems
- MQTT/Kafka messaging
- CCTV feeds
- Weight sensors
- BLE devices
